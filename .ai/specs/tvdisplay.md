# Spec: TV Display

**Status:** Live

---

## What TV Display Does

A real-time performance dashboard designed to run on a TV screen in the office. Shows live agency performance data to motivate agents and give principals visibility without opening reports.

---

## Current Features

- Live agent deal leaderboard
- Sales and rental transaction counts
- Commission totals (period-selectable)
- Listing stock counts
- Scrolling ticker / highlight cards
- Auto-refreshes without manual intervention
- Branded to agency

---
