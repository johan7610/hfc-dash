# 10 — Quick Reference Cards (Printable)

**Print this document and keep it at your desk.**

---

## System Owner — Daily Cheat Sheet

### Your Main Destinations
1. **System Developer > Agency Management** — manage agencies
2. **System Developer > Importer** — bulk imports from P24
3. **System Developer > Fault Reports** — system issues
4. **System Developer > Feedback Reports** — staff feedback
5. **Admin > Settings** — agency-wide configuration

### Top 5 Actions
1. Check Fault Reports for new system issues
2. Review Feedback Reports from staff
3. Monitor agency activity via Agency Management
4. Import new agents or listings via the Importer
5. Update system settings as needed

### Common Gotchas
- The Agency Switcher is at the top of the sidebar. Select an agency before doing anything agency-specific.
- Permanent agency deletion requires typing the confirmation password: you only get one chance.
- The P24 Importer is a multi-step process — agents must be imported before listings.

### If You're Stuck
First thing to try: refresh the page. Second: check you've selected the right agency in the switcher.

## Administrator — Daily Cheat Sheet

### Your Main Destinations
1. **Today** — your action cards (landing page)
2. **Admin > Onboarding** — new agent pipeline
3. **Admin > User Management** — activate/deactivate users
4. **Admin > Settings** — configure everything
5. **Compliance > FICA** — review compliance queue

### Top 5 Actions
1. Check Today page for overdue items and pending approvals
2. Process onboarding applications (verify docs, advance pipeline)
3. Review and approve FICA submissions
4. Manage user accounts (new users, deactivations, role changes)
5. Monitor compliance reporting queue

### Common Gotchas
- Onboarding **"Activate"** won't work until all required checklist items are complete.
- When deactivating a user, their data is preserved — nothing is deleted.
- Settings has 12 sections. Use the tabs at the top to navigate between them.
- Commission routes don't appear in the sidebar for all roles — they're under Agency Tracker > Commission.

### If You're Stuck
First thing to try: switch to the user's view (impersonate) to see what they see. Use the user picker at the bottom of the sidebar.

## Branch Manager — Daily Cheat Sheet

### Your Main Destinations
1. **Today** — management cards (B1-B5)
2. **Dashboard > Oversight** — items needing attention
3. **Dashboard > Branch Report** — branch performance
4. **Leave Management > Applications** — approve/reject leave
5. **Agency Tracker > Branch > Branch Performance** — branch metrics

### Top 5 Actions
1. Check Today page — focus on the Branch Agent Watch and Compliance Queue cards
2. Review Oversight feed and send nudges where needed
3. Approve/reject leave applications
4. Approve deal pipeline steps that are awaiting your sign-off
5. Review branch performance and daily activity summary

### Common Gotchas
- You can only see data for your own branch (unless you have cross-branch permissions).
- Deal step approvals appear on your Today page as "Deal Steps Assigned to Me".
- Leave rejection requires a reason of at least 10 characters.
- TV Messages are per-branch. Your messages only appear on your branch's screens.

### If You're Stuck
First thing to try: check Oversight. If the issue is with a specific agent, click their name in Oversight to see their activity.

## Agent — Daily Cheat Sheet

### Your Main Destinations
1. **Today** — your 19 action cards (landing page)
2. **Real Estate > Properties** — your listings
3. **Real Estate > Contacts** — your clients
4. **Documents > My E-Sign Documents** — document tracking
5. **Dashboard > Calendar** — your schedule

### Top 5 Actions
1. Check Today page — action any overdue items first
2. Log daily activity: Agency Tracker > My Daily Activity
3. Check calendar for today's viewings and appointments
4. Follow up on e-sign documents awaiting signatures
5. Update properties that need attention (photos, details, FICA)

### Common Gotchas
- **Marketing Readiness has 4 gates** — all must pass before you can publish. The Readiness Panel on the property page shows which are blocking.
- Sale agreements (OTPs) **cannot use e-sign** — they must be signed with wet ink. The system will tell you.
- When adding a contact, set the correct Contact Type (Seller, Buyer, etc.) — it affects how the system matches them later.
- Core Matches need criteria saved before results appear. If results are empty, broaden your search.

### If You're Stuck
First thing to try: check the Today page — it usually shows you what needs your attention. If a feature seems missing, ask your branch manager about your permissions.

## Compliance Officer — Daily Cheat Sheet

### Your Main Destinations
1. **Compliance > FICA** — FICA review queue (check My CO Queue tab)
2. **Compliance > Verification Queue** — staff document verification
3. **Compliance > Compliance Reporting** — whistleblower complaints
4. **Compliance > RMCP Dashboard** — RMCP acknowledgement progress
5. **Compliance > Staff Screening** — background checks

### Top 5 Actions
1. Process FICA queue — approve, return, or reject submissions
2. Verify documents in the Verification Queue
3. Review and action compliance complaints (approve sends to PPRA)
4. Monitor RMCP acknowledgement progress and send reminders
5. Check for overdue staff screenings

### Common Gotchas
- FICA approval requires drawing your signature on the pad — it must be drawn, not just clicked.
- **"Reopen for Corrections"** is a new feature — use it instead of creating a new FICA request when a rejected one needs another chance.
- Verification Queue badge count refreshes every 60 seconds. Refresh the page if it seems stale.
- Compliance Reporting is currently in **demo mode** — PPRA emails go to a test address, not the real PPRA.
- When rejecting a FICA, the rejection is permanent unless reopened. Double-check before clicking.

### If You're Stuck
First thing to try: check the Communications Log (Compliance > Communications Log) to verify whether emails were sent. If a FICA submission is missing from your queue, switch to the "All" tab.

## Keyboard Shortcuts & Navigation Tips

| Action | How |
|--------|-----|
| Go to Today page | Click **"Dashboard > Today"** or the CoreX logo |
| Switch between sidebar groups | Click the group name to expand/collapse |
| Go back in a submenu | Click the **"Back"** button at the top of the submenu panel |
| Toggle dark/light mode | Click the sun/moon icon next to your name at the bottom of the sidebar |
| Search for a property | Use the search box on any property picker — type at least 2 characters |
| Search for a contact | Use the search box on any contact picker — type at least 2 characters |

## Emergency Quick Actions

| I need to... | Go to... |
|-------------|---------|
| Cancel a document that was sent for signing | Documents > My E-Sign Documents > find it > **"Cancel"** |
| Reopen a rejected FICA | Compliance > FICA > find the rejected submission > **"Reopen for Corrections"** |
| Send a signing reminder | Documents > My E-Sign Documents > find it > **"Send Reminder"** |
| File a compliance complaint immediately | Compliance > Compliance Reporting > **"File New Report"** |
| Apply for leave urgently | My Portal > Leave > **"Apply"** |
| Report a system problem | Click the help icon in the sidebar header, or go to the Feedback Reports page |
